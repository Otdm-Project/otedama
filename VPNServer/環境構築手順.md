# VPNServer環境構築手順
## WireGuardのインストール
```
sudo dnf install -y epel-release
sudo dnf install -y wireguard-tools
```
## Rustのインストール
```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source $HOME/.cargo/env
sudo dnf install -y gcc
```

## Rustプロジェクトの作成
```
cargo new vpnserver
cd vpnserver
```
## pyrhonのインストール
```
sudo dnf install python3-pip -y
pip install cqlsh
```
